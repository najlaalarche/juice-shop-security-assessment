# Project Scope

This project is a beginner-level cybersecurity assessment conducted for educational purposes.

## In Scope
- Manual analysis of the OWASP Juice Shop web application
- Observation of user input handling
- Inspection of HTTP requests and responses using browser developer tools
- Identification of common web security risks at a conceptual level

## Out of Scope
- Attacking real-world websites or services
- Automated scanning tools
- Denial of Service attacks
- Data destruction or modification
- Exploitation beyond learning objectives
